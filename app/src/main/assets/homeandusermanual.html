By this application you can get the all emergency data all over Bangladesh. By Proceeding some steps.

User Manual

  • 1st you have to select your location where you want to get data.
  • Then you can select the category to find which you want.
  • You can also find the category by tapping the private or public tab.
  • You can see the google map of that place which you want.
  • You can make call, message and also can share the application to friends also.
  • Also you can find the data outside your city by again selecting the location.
